## Basic Set

To create a basic set, you will want a selection that lets you build a few small rooms, or a larger, more complex room. To that end, we suggest you start off with the following:

* 14 walls
* 8 corners
* 12 2x2 floors
* 2 4x4 floors
* 4 doors
* 2 wide doors
* 2 windows
